# GiteePages

#### 介绍
Gitee Pages 是一个免费的静态网页托管服务，您可以使用 Gitee Pages 托管博客、项目官网等静态网页。
已开启 Gitee Pages 服务，网站地址： http://liuyandeng.gitee.io/giteepages

文档:https://gitee.com/help/articles/4136

#### 注意
上传完静态文件后,要在Gitee Pages 服务中点击"更新"按钮
